# Supabase シードデータ投入手順（Chrome Agent用プロンプト）

## ステップ1: Supabaseにログイン

1. https://supabase.com にアクセス
2. ログイン（GitHubアカウント）
3. プロジェクト一覧からMachibuseプロジェクトを選択

## ステップ2: SQL Editor を開く

1. 左メニューから「SQL Editor」をクリック
2. 「New query」をクリック

## ステップ3: シードデータを実行

## ステップ4: Realtime を有効化

1. 左メニューから「Database」→「Replication」をクリック
2. 「Tables」セクションで「notifications」テーブルのトグルを ON にする
3. これにより、通知のリアルタイム配信が有効になる

## ステップ5: 確認

1. https://machibuse.vercel.app/mansions にアクセス
2. 7件の建物が表示されれば成功
3. 各建物をクリックして、間取りタイプや募集情報が表示されることを確認
